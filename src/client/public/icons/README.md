Put your app icons in this folder:

- `icon-192.png` - manifest icon, 192x192
- `icon-512.png` - manifest icon, 512x512
- `maskable-512.png` - manifest maskable icon, 512x512
- `favicon-32.png` - browser favicon, 32x32
- `apple-touch-icon.png` - Apple touch icon, 180x180 recommended

These files are served from:

- `/icons/icon-192.png`
- `/icons/icon-512.png`
- `/icons/maskable-512.png`
- `/icons/favicon-32.png`
- `/icons/apple-touch-icon.png`
